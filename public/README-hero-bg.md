# Hero Background Image

Please save your futuristic space/technology background image as:
`hero-bg.jpg`

This file should be saved in the /public folder.

The image will be used as the background for the hero section with the following properties:
- Background size: cover
- Background position: center
- Background repeat: no-repeat
- Overlay: Semi-transparent black (bg-black/60) for better text readability

Image dimensions recommendation: 1920x1080 or higher for best quality.
